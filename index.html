Welcome to the tiles memory game!

What is a memory game?

The memory game is a series of objects which have matching pairs. You may only reveal 2 object at a time as they will initially be hidden. If the two objects match, you can then proceed in trying to find the next matching pair. If both objects do not match, then you must hide the objects once again but try to remember where they are for later. Once you have found all matching pairs, you will have completed the game.
